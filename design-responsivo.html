Design Responsivo (ou responsividade)

Conjunto de técnicas de desenvolvimento fornt-end visando a construção de sites em que o layout e a experiência geral de uso funciona bem em dispositivos com telas de tamanhos variados.

Existem duas abordagens para desenvolvimento responsivo:

Mobile First (é o mais usado)

Nesta abordagem o desenvolvimento é focado para primeiro em telas pequenas, para somente depois de realizarmos ajustes para telas maiores.

Desktop First
